## Desktop App for All

### Features

* Localstorage
* AngularJS, Nodewebkit

### Usage

```sh
# Open browser
grunt serve
```

```sh
# Build 
grunt buildApp
```

```sh
# Run application 
grunt runApp
```

```sh
# Build and Run application 
grunt buildAndRunApp
```
